Git branches are incredibly lightweight. A branch in Git is simply a lightweight movable pointer to one of these commits.

The default branch name in Git is `master` (or `main` in modern repositories). As you start making commits, you're given a master branch that points to the last commit you made. Every time you commit, the pointer moves forward automatically.
